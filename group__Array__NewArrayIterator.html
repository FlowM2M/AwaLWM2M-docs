Detailed Description

Allocate and return a pointer to a new Array Iterator of the specified type, that can be used to walk through values inside the array.

The Array Iterator is owned by the caller and should eventually be freed with Awa___ArrayIterator_Free.

Parameters
[in]arrayA const pointer to a valid resource array of the specified type.
Returns
the newly allocated Array Iterator on success.
NULL on failure.
