Detailed Description

Determine whether a resource instance at the specified index of an array is valid.

Parameters
[in]arrayA pointer to a valid resource array of the specified type.
[in]indexThe index of the resource instance to retrieve.
Returns
true if the input array is valid and the resource instance at the specified index is not NULL
false if the input array is invalid, the index is out of range, or the value at the specified index is NULL.
